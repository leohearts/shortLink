# shortLink
A simple short link generator (PHP, SQLite3)

## Install

- Install `php-sqlite3`
- Copy all files to your web directory
- Give www-data write access to the directory
- Enable .htaccess file
    - `a2enmod rewrite`
    - `AllowOverride All` in `/etc/apache2/apache2.conf`
